How Links Get Built

Affiliate Link Resolution — Priority Order

For every product the AI recommends, the pipeline resolves the best affiliate link using this waterfall. The goal: always use a tracked, Steph-branded link.

1
URLGenius Short Link
If this product has a URLGenius link → use it. Already tracked, has product image, click data accumulates.
→
amzn/zvN_M9
✓ Best — tracked + click data
2
Walmart Direct API
For Walmart products (especially storefront items): use Walmart API to generate affiliate link with Creator tag.
→
walmart.com/ip/[id]?wmlspartner=...
✓ Live price + in-stock verification
3
Amazon ASIN + Affiliate Tag
For Amazon products: build standard affiliate URL from ASIN. Simple and always works.
→
amazon.com/dp/[ASIN]?tag=mommymedeals-20
✓ Works for all 35,929 ASINs
4
Impact-Generated Link (Target)
For Target products: use Impact API to generate tracking link. Less common but covered.
→
goto.target.com/c/[id]/...
5
SYL URL (Other Retailers)
For Wayfair, Ulta, Kohl's, Kids Foot Locker, etc.: use the SYL-encoded URL (already has affiliate tracking built in).
→
wayfair.com/...[SYL-tracked]
✨
Fallback — LTK Storefront
If no specific link is known: always fall back to Steph's LTK page. Covers everything she's curated.
→
shopltk.com/EverydaywithSteph
Always works · universal fallback
