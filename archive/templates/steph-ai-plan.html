Key Technical Decision
Two modes: (1) Reactive — AI answers visitor questions. (2) Proactive — A weekly Python script processes all CSVs and generates a "Top Products" knowledge block that gets injected into the AI's context automatically.
No database needed for Phase 1 — the weekly CSV exports become the knowledge base. Just refresh the file.
Build Roadmap

3-Phase Plan

1

Phase 1 — Smart Chatbot (Done + Upgrade)

Now — 1 Week

Take the chatbot we already built and supercharge the knowledge base with real data from all 6 sources.

  • Load processed CSV data into the system prompt knowledge block
  • Add a weekly Python script that auto-generates "Top Products" from the data
  • Inject real product names, prices, links, and click counts into AI context
  • Add suggested questions: "What's hot this week?" "Best under $20?" "What's selling at Walmart?"
2

Phase 2 — Content Generator Tab

Next — 2–3 Weeks

Add a second mode to the UI: "Generate Content" — lets Steph's team produce captions, top 5 lists, and deal alerts on-demand.

  • Tab 2: "Create Content" — select format (Top 5, Caption, Deal Alert)
  • AI pulls from live data to draft in Steph's voice
  • One-click copy to clipboard for Facebook/Instagram posting
  • Weekly auto-generated "This Week's Hot Picks" email/post draft
3

Phase 3 — Live Data Pipeline + Dashboard

Later — 1–2 Months

Connect data sources via API instead of CSV exports. Real-time trending. Performance dashboard for Steph's team.

  • Amazon Product Advertising API → live product data + prices
  • Automated weekly report email: "Here's what sold, what clicked, what to post"
  • Admin dashboard: swap in new deals, toggle affiliate links, preview AI responses
  • Optional: visitor data collection for personalization ("you looked at toddler toys last week")
